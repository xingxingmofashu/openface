<Info>
  **Prerequisites**
  - Bun installed for the CLI project
  - Mint installed globally for the docs preview
</Info>

## Project commands

<Steps>
<Step title="Install dependencies">

```bash
bun install
```
</Step>

<Step title="Run the CLI locally">

```bash
bun dev
```

This starts the OpenFace CLI entrypoint with Bun from the repository root.

</Step>
<Step title="Type-check the workspace">

```bash
bun typecheck
```
</Step>
</Steps>

## Preview docs

From the `docs` directory:

```bash
mint dev
```

Mint will start a local preview, usually on `http://localhost:3000`.

## Validate docs

```bash
cd docs
mint validate
```

## Useful checks

- `mint validate` checks the docs build in strict mode.
- `mint broken-links` scans internal and external links.
- `bun typecheck` verifies the TypeScript workspace.

## Repository layout

```text
packages/openface/src/
├── cli/
│   ├── commands/     # yargs command modules
│   └── utils/        # REPL and helper utilities
├── config/           # Global config loading and defaults
├── tasks/            # Hugging Face pipeline wrappers
└── index.ts          # CLI entrypoint
```

## Docs files

The Mint site for this repository lives under `docs/` and is configured through `docs/docs.json`.

## Notes

- The root `package.json` exposes `bun dev` and `bun typecheck`.
- The current command UX is REPL-first for `translation` and `text-generation`.
- Model downloads and cache behavior are documented in the configuration reference.

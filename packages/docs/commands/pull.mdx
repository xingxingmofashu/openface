---
title: "pull"
description: "Download models, tokenizers, and processors from Hugging Face Hub for a specific task."
---

## Synopsis

```bash
openface pull --model <model-id> --task <task>
```

## Options

| Flag | Required | Description |
| --- | --- | --- |
| `--model` | Yes | Hugging Face model repository ID. |
| `--task` | Yes | Task type used to decide which model, tokenizer, or processor artifacts to download. |

## Common examples

```bash
openface pull --model Helsinki-NLP/opus-mt-zh-en --task translation
openface pull --model Xenova/gpt2 --task text-generation
openface pull --model Xenova/distilbert-base-uncased-finetuned-sst-2-english --task text-classification
```

## What gets downloaded

Depending on the task, OpenFace may pull:

- a tokenizer
- a processor
- one or more model files

The implementation resolves these requirements from `SUPPORTED_TASKS` in `packages/openface/src/tasks/pull/tasks.ts`.

## Supported task categories

The CLI currently supports a broad set of task names, including:

- `translation`
- `text-generation`
- `text-classification`
- `token-classification`
- `summarization`
- `question-answering`
- `feature-extraction`
- `automatic-speech-recognition`
- `text-to-audio`
- `image-classification`
- `image-segmentation`
- `object-detection`

Run `openface pull --help` to see the full list exposed by the current build.

## Cache behavior

Downloaded artifacts are stored in the configured cache directory. By default that is:

```text
~/.local/share/openface/models
```
